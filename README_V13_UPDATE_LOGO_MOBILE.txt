# AtariHelp APK Builder v13 - Mobile native + Logo

Tahle verze už není kopie webu přes iframe.

## Co přidává
- N&P logo v appce
- N&P logo jako ikona aplikace
- mobilní domovskou stránku AtariHelp centra
- Hry / Návody / N&P sekce jako mobilní obsah
- PiTT-KiTT už není úvodní stránka, je v sekci Hry
- XEX runner zůstává, ale je poctivě označený jako nedokončená emulace

## Aktualizace na GitHubu
1. Rozbal ZIP.
2. Otevři svůj GitHub projekt `atarihelp-apk`.
3. Code → Add file → Upload files.
4. Přetáhni obsah rozbalené složky.
5. Commit changes.
6. Actions → počkej na zelenou fajfku.
7. Stáhni HOTOVE_APK_ATARIHELP_STAHNI_ME.
8. Rozbal a nainstaluj nové app-debug.apk.

Před instalací nové verze odinstaluj starou AtariHelp.
