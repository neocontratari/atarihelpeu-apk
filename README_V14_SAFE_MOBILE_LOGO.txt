# AtariHelp APK Builder v14 SAFE mobile + logo

Tahle verze je bezpečná oprava po pádu buildu v13.

Důležité:
- Nemaž staré verze ani staré workflow.
- Chyba nebyla tím, že jsi odmazal v3/v4.
- v14 vychází ze známého funkčního v11 Android základu.
- Android část je minimálně změněná, aby build zase prošel.
- Logo je uvnitř appky.
- Ikona aplikace je bezpečná XML N&P/gold varianta, ne rizikový PNG resource.

Jak aktualizovat:
1) Rozbal ZIP.
2) GitHub → atarihelp-apk → Code.
3) Add file → Upload files.
4) Přetáhni obsah rozbalené složky.
5) Commit changes.
6) Actions.
7) Počkej na zelenou fajfku.
8) Stáhni HOTOVE_APK_ATARIHELP_STAHNI_ME.
9) Rozbal a nainstaluj app-debug.apk.

Když build spadne, klikni na červený běh → Vyrob debug APK → rozbal krok 6 - Vyrobit APK a pošli mi červený text.
