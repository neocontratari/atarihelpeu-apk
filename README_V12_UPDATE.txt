# AtariHelp APK Builder v12

Update pro existující GitHub projekt.

## Co přidává v12

- záložku Web
- pokus o listování AtariHelp.eu přímo v appce
- fallback tlačítko Otevřít přímo
- PiTT-KiTT XEX zůstává uvnitř
- mini runner/debug základ zůstává uvnitř

## Jak aktualizovat

1. Rozbal tento ZIP.
2. Na GitHubu otevři svůj projekt atarihelp-apk.
3. Klikni Code.
4. Klikni Add file → Upload files.
5. Přetáhni celý obsah rozbalené složky.
6. Dole klikni Commit changes.
7. Klikni Actions.
8. Počkej na zelenou fajfku.
9. Stáhni artifact HOTOVE_APK_ATARIHELP_STAHNI_ME.
10. Rozbal ZIP a nainstaluj nové app-debug.apk.
