ZAČNI TADY – AtariHelp APK

Tohle je klikací návod pro výrobu opravdového APK přes GitHub. Žádný Android Studio, žádná příkazová řádka.

Důležité

Nenahrávej na GitHub ten ZIP. Nejdřív ho rozbal. Na GitHub se nahrává obsah rozbalené složky.

1) GitHub

Otevři github.com, přihlas se, klikni vpravo nahoře na + a dej New repository.

Název: atarihelp-apk
2) Vytvoř projekt

Nedávej README. Klikni Create repository.

3) Rozbal ZIP

Rozbal tenhle ZIP. V rozbalené složce musíš vidět:

.github app build.gradle settings.gradle README.md
4) Nahraj soubory

Na GitHubu klikni Add file → Upload files. V rozbalené složce dej Ctrl+A a přetáhni všechno do GitHubu.

5) Ulož

Dole klikni Commit changes.

6) Vyrob APK

Klikni Actions → VYROBIT APK ATARIHELP. Když to neběží samo, dej Run workflow.

7) Stáhni APK

Až bude zelená fajfka, otevři běh a dole stáhni artifact:

HOTOVE_APK_ATARIHELP_STAHNI_ME
8) Instaluj do mobilu

Rozbal stažený artifact. Uvnitř je:

app-debug.apk

Ten nainstaluj do mobilu.
