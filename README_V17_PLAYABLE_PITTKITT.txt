# AtariHelp.eu APK Builder v17 - Playable PiTT-KiTT mobile port

## Co je nové

- přibalený zdroják PiTT-KiTT ASM v assets/sources
- přibalený oficiální PDF manuál v assets/docs
- náhledy stránek manuálu v mobilní sekci Manuál
- mobilní katalog pro HRY / XEX / WAV / PDF podle dodaných odkazů
- první hratelný mobilní port PiTT-KiTT
- XEX technický runner zůstává zvlášť
- název aplikace zůstává AtariHelp.eu
- ikona aplikace používá N&P logo jako app_icon.png

## Důležité

Tohle není plná Atari XEX emulace.
Je to první hratelný mobilní port podle zdrojáku a manuálu.

Ovládání:
- LEFT / RIGHT = řízení
- FIRE = střela
- JUMP = turbo skok přes díry
- 10 levelů podle manuálu
- skóre = zničení soupeři

## Aktualizace na GitHubu

1. Rozbal ZIP.
2. GitHub → atarihelp-apk → Code.
3. Add file → Upload files.
4. Přetáhni obsah rozbalené složky.
5. Commit changes.
6. Actions.
7. Stáhni HOTOVE_APK_ATARIHELP_STAHNI_ME.
8. Rozbal a nainstaluj app-debug.apk.

Před instalací nové verze odinstaluj starou AtariHelp.eu.
