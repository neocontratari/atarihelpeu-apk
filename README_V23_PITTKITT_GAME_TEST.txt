# AtariHelp.eu v23 - PiTT-KiTT samostatný herní test

Vychází z funkční v22.

## Co je změněné
- zachované funkční menu z v22
- žádná přestavba aplikace
- přidán samostatný soubor pittkitt_game.html
- přidána karta Herní test v sekci Hry
- hra je oddělená od menu

## Test
1. Otevři appku.
2. Ověř, že menu pořád funguje jako ve v22.
3. Jdi do Hry.
4. Klikni Hrát PiTT-KiTT test.
5. Zkus LEFT / RIGHT / FIRE / JUMP.
6. Tlačítkem Zpět se vrať do aplikace.

Tohle je mobilní test/port, ne plná Atari XEX emulace.
