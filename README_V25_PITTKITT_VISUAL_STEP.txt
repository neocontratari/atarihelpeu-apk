# AtariHelp.eu v25 - PiTT-KiTT visual step

Vychází z funkční v24.

## Co je změněné
- menu zůstává stejné
- upravený pouze pittkitt_game.html
- lepší KITT auto
- červený scanner
- vylepšená silnice
- level banner
- lepší nepřátelská auta
